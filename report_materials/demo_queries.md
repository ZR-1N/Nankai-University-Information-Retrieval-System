## 八、索引中不存在日期的说明

当前索引中的日期格式为 `YYYY-MM-DD`（如 2026-06-12），jieba 分词按字符分割，`2026-06-12` 作为一个整体 token 存入索引，而 `2026年06月12日` 会被拆分为 `2026年`+`06月`+`12日`。

因此 `20??年`、`202?-06` 等日期通配符在当前索引中返回 0 —— 这些模式的 token 不存在于索引词典中。

**如果需要日期搜索，建议使用普通查询模式搜索年份数字（如 `2026`），而非通配符。**
